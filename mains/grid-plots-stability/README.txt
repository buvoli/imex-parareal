This directory contains the main files and functions for generating the parareal stability figures

-- Script Descriptions -------------------------------------------------------------------------------------------------
The following scripts where run for this paper:

SCRIPT:         grid_script_paper.m
DESCRIPTION     Generates high-resolution grid figures that are contained in paper writeup.

SCRIPT:         grid_script_all.m
DESCRIPTION:    Generates low-resolution grid figures for a wide range of parareal parameters.

SCRIPT:         single_plot.m
DESCRIPTION:    Generates the three example figures for explaining convergence, stability, and overlay figures.

-- Directory Descriptions ----------------------------------------------------------------------------------------------

DIR:            figures
DESCRIPTION:    contains all figures outputs
